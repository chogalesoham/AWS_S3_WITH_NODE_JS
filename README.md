# AWS S3 with Node.js

Simple Node.js project demonstrating **file upload to AWS S3** using **Multer** and **MongoDB**.

## Tech Stack
- Node.js
- Express.js
- AWS S3
- Multer
- MongoDB

## Features
- Upload files to AWS S3
- Store file metadata in MongoDB
- Secure and scalable storage

## Use Case
Best for apps needing image/file uploads like **e-commerce**, **profiles**, or **documents**.
